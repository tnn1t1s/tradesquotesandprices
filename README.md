# Trades, Quotes and Prices Examples

This repository contains Python implementations of examples from the book "Trades, Quotes and Prices" by Jean-Philippe Bouchaud et al.

## Setup
1. Clone the repository
2. Create virtual environment: \`python -m venv venv\`
3. Activate virtual environment: 
   - Unix/MacOS: \`source venv/bin/activate\`
   - Windows: \`.\venv\Scripts\activate\`
4. Install dependencies: \`pip install -r requirements.txt\`

## Structure
- \`examples/\`: Implementation of book examples
- \`data/\`: Data files used in examples
- \`notebooks/\`: Jupyter notebooks for interactive exploration

## Book Reference
Bouchaud, J. P., Bonart, J., Donier, J., & Gould, M. (2018). Trades, quotes and prices: financial markets under the microscope. Cambridge University Press.
